Điểm nổi bật

Không chỉ là chatbot SQL. Đây là một workflow dữ liệu hoàn chỉnh.

Natural Language to SQL

Đặt câu hỏi bằng tiếng Việt, để hệ thống đọc schema, hiểu ngữ cảnh và tạo truy vấn phù hợp thay vì bắt bạn viết SQL từ đầu.

Plan Approval

Mỗi truy vấn có thể đi qua bước xem trước kế hoạch, giúp giảm rủi ro chạy sai ý hoặc thao tác nhầm trên dữ liệu thực tế.

Cross-Database Context

Chọn nhiều database trên cùng một server MySQL để AI có thể suy luận và truy vấn xuyên nguồn dữ liệu.

Session Memory

Mỗi cuộc hội thoại được lưu lại kèm SQL, dữ liệu thô và cấu hình biểu đồ để bạn quay lại đúng mạch làm việc trước đó.

Visual Output

Khi phù hợp, app tự sinh biểu đồ để biến kết quả truy vấn thành tín hiệu dễ đọc hơn cho báo cáo và phân tích nhanh.

Desktop Native Packaging

Gói sẵn cho Windows, macOS và Linux, không cần dựng backend riêng trong luồng sử dụng cuối cùng.

Tải xuống

Chọn đúng bản cài đặt cho từng phiên bản và nền tảng.
